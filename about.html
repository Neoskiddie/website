About this website

The website began as a university project during my first year - a one-page portfolio. I've kept changing it afterwards multiple times. Currently, this is a fourth complete rewrite. The design, not the implementation, was always the biggest challenge.

Now it runs on MEAN stack (without Angular). The front end is mostly designed and created using Bootstrap and a bit of Jquery. But to avoid hosting it on a server I'm just taking static snapshots of the website and publishing it to GitHub. That way I don't need to worry about updating the website, paying for the server etc. You can read more about this website here
